### 시즌1: '이것이 취업을 위한 코딩 테스트다 with 파이썬' 책 완독

- 숫자 이모지는 책에서 나온 문제의 난이도(1~3)를 의미합니다. ◻: +0.5
- 색깔 이모지는 백준에서 나온 문제의 난이도를 의미합니다.
- 성공 여부는 문제를 목표 시간 내에 해결했는가에 달려 있습니다.
